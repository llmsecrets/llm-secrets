Compatible passkey providers ▶
iCloud Keychain Recommended for iOS
Built-in on iPhone, iPad, and Mac. No app to install.
Requires iOS 18+ / Safari 18+.
Settings → Passwords → Password Options → ensure iCloud Passwords is on.
Google Password Manager Recommended for Android
Built-in on Android. No app to install.
Requires Android 14+ with Chrome or the default browser.
Settings → Passwords & Accounts → ensure Google is the default passkey provider.
Bitwarden Free · iOS & Android
Free, open-source password manager with FIDO2/passkey support.
Works on both iOS and Android. Great fallback if iCloud Keychain or Google Password Manager doesn't work.
iOS: Settings → Passwords → Password Options → enable Bitwarden.
Android: Settings → Passwords & Accounts → set Bitwarden as default.
1Password Paid · iOS & Android
Premium password manager with full PRF support.
Works on both iOS and Android. Setup is the same as Bitwarden — set it as your passkey provider in system settings.
